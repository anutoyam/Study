HTML - 정보, 전자출판을 하려 만듬 *어떻게 정보를 잘 표현할 것인가.

[디자인의 필요성] --> 정보를 떠나서 웹페이지가 아름다웠으면 함.
->
디자인 태그를 넣기 시작.
->
<font> --> 디자인 대표 태그
->
<Li><font color = "red">HTML</font></Li>
->
Li, font 태그가 공존한다
Li = 정보를 담는 태그
font = 시각적 디자인을 담는 태그

정보와 디자인이 믹스되버리기 시작하자. 
html에 디자인적 기능을 빼버림(html이 정보에 전념했으면 함)
->
디자인을 담당하는 언어인 CSS를 만듬
->
<style>
이 사이에 들어오는 태그는 html이 아니라 디자인 태그다
Li{
    color:red;
}
Li가 들어가는 모든 언어는 빨간색 글씨로 바뀜
</style>
<Li>HTML</Li>

-> 
정보와 디자인이 분리된다.
